# ðŸ“‹ Criteria
* GIVEN I am taking a code quiz
* WHEN I click the "Start Quiz" button
* THEN a timer starts, and I am presented with a question
* WHEN I answer a question
* THEN I am presented with another question
* WHEN I answer incorrectly
* THEN time is subtracted from the clock
* WHEN all questions are answered OR the timer reaches 0
* THEN the game ends
* WHEN the game is over
* THEN I can save my initials and score

## ðŸ“š Usage
1. Open the JavaScript Quiz in your browser.
2. Click the "Start Quiz" button to begin.
3. Read each question carefully and select an answer.
4. Correct answers add to your score; incorrect answers deduct time.
5. When the quiz ends, enter your initials to save your score.
